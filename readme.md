# Food Delivery App

> Food delivery application built with Node.js Express MongoDB for backend, React Redux Material-UI for frontend

## Quick Start
Add .env file to the server folder. Make sure you set an env each for PORT, MONGODB_URI, JWTSECRET

```bash
# Install dependencies for frontend
npm install

# Install dependencies for backend
cd server && npm install

# Run the server & frontedn with concurrently(in the root folder)
npm run dev

# Build the app frontend for production to the build folder
npm run build

# Run the app in production mode
npm start
```
